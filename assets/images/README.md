# Images

Drop your images in this folder. Reference them from any HTML file using
a path like `assets/images/your-image.jpg` (or `../../assets/images/...`
when you're inside a sub-page).

A few suggestions:

- Use `.jpg` for photos, `.svg` for logos and icons, `.png` only when you
  need transparency.
- Keep images under ~500KB for fast loads. Resize to 1200px wide max.
- Use descriptive filenames: `morning-walk.jpg`, not `IMG_4837.jpg`.
