The Garden

A digital garden is a metaphor for thinking in public. Like a physical garden, what matters is the tending: planting ideas, watering them with reading, pruning them through revision.

Where a blog is a public diary — chronological, finished, tidy — a garden is a personal laboratory in progress. Notes here grow non-chronologically and branch outward. Some pages are developed; others are barely sketches. Come back later and you might find something changed, or linked somewhere new.

There's no prescribed path through this garden. Start anywhere and let the links guide you.

How to add a new note

Make a new folder inside en/garden/. The folder name becomes the URL — keep it lowercase and hyphenated, like en/garden/the-quiet-page/. Copy first-note/index.html into it as a starting point, then edit. Don't forget to link to it from this page.
