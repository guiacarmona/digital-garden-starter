Your first note: a starting point

Welcome to your first note. This page exists to show you the shape of a note in this garden. Copy this folder, rename it, and start writing.

What a note can be

A note can be a half-formed thought you want to come back to. A short essay. A list. A question. A quote with a few lines of commentary. The point is that it's yours, and it doesn't have to be finished.

A garden grows non-chronologically and branches outward. Where a blog is a public diary, a garden is a personal laboratory in progress.

Linking between notes

Use ordinary links to connect notes — that's how the garden's network forms over time. For example, you might want to link back to the garden index, or out to another note that develops the same idea.

Note status

I find it useful to mark each note's stage at the top — seedling, growing, evergreen. Readers know what they're getting, and you give yourself permission to publish unfinished things.
